How to Use the Tatkal Ticket Booking Extension

Fill all the form Login , Train and Payment Details

  1. Fill in your login details. Don't worry, your information is safe and stored securely in your Chrome local storage.
  2. Enter the details of the trains you want to book Tatkal tickets for. You can use the Go to IRCTC Website button to find the required information and fill it in the form.
  3. We provide default payment options via Paytm UPI, which is faster compared to other payment methods.
  4. If you want to see the final QR code page for payment, check the box labeled Pay & Book (Show QR Code Page).
  5. Timer Details:
    1. Tatkal Start Timer:
      • If you select AC class, Tatkal start timer will be set for 09:59:45.
      • If you select Sleeper class, Tatkal start timer will be set for 10:59:45.
      • We recommend using the default timer settings.
    2. Refresh Time (ms): Default is set to 5000 milliseconds (5 seconds). This will refresh the train availability status every 5 seconds on the IRCTC page.
    3. You can adjust both the start timer and refresh time according to your preferences and needs.
  6. The tatkal start timer will be used to transition from search train page to train list page and there we are going to refresh the selected train every 5 seconds
  7. Once you've filled all the details, click on the Save Settings button.

Auto Booking Switch

  1. If the switch is turned on, the extension will run automatically whenever you visit the IRCTC website and attempt to book tickets using the provided settings and passenger details.
  2. If the switch is turned off, the extension will not run on the IRCTC page.
  3. If the switch is on and Tatkal start timer is set, the extension will automatically log in 2 minutes before the start timer, and the Go to IRCTC Website button will disappear.
  4. If the switch is on and Tatkal start timer is set, and there is less than 2 minutes available before the start timer, the Book Ticket on IRCTC button will appear, allowing you to book manually.

How to Add a New Passenger

  1. Fill in the passenger's details including name, age, gender, and preference.
  2. Click on the Add Passenger button.
  3. The newly added passenger will be automatically selected for Tatkal ticket booking.
  4. If you don't want to include a passenger in the booking, you can uncheck or delete them.

What to Do on the IRCTC Site While Booking?

  1. At the login prompt, fill in the captcha and press Enter on your keyboard.
  2. At the Review and Captcha Page, a prompt will appear displaying the Current Seats Status: . Use this information to decide whether to proceed with booking the ticket. If you decide to proceed, fill in the captcha shown on the page in the prompt, then press Enter on your keyboard.
